Question and Answer

1) What is DOM and what is the purpose of DOM?

Ans: DOM means Document Object Model.It represents the web interface.
Main purpose of DOM is to change the document structure, style, and content.

2) How will you select HTML elements using DOM? Name the DOM methods.

Ans: In DOM API there are some dom methods which we can use to call HTML elements by their id, class, tag.
Structure: document.getElementById("myElement");
There are 5 types of DOM methods with whom we can call HTML elements.They are:
1.getElementById.
2.getElementByClassName.
3.getElementsByTagName.
4.querySelector.
5.querySelectorAll.

3) What is event bubble?

Ans: Event Bubble is a concept in the DOM that happens when an element receives an event, and that event bubbles up to its parent elements in the DOM tree until it gets to the root element.

4) What is a callback function and why will you use it?

Ans: Callback function is a function which passes it's data to other functions when it's called by the parent function.
Call back function has many uses.One of the important use is- If there any argument which we have to use in multiple functions then a call back function is declared to implement the reusable codes.
